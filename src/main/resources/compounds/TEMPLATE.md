# 化合物文件模板

## 文件格式说明
在 `src/main/resources/compounds/` 目录下创建 `.txt` 文件来定义化合物。

## 文件格式
```
名称:化合物的中文名称
化学式:化学式
摩尔质量:数值（单位：g/mol）
状态:固体/液体/气体
描述:化合物的描述
```

## 示例1：氢气
```
名称:氢气
化学式:H2
摩尔质量:2.016
状态:气体
描述:无色无味的气体，可燃
```

## 示例2：氯化钠
```
名称:氯化钠
化学式:NaCl
摩尔质量:58.443
状态:固体
描述:白色晶体，食盐的主要成分
```

## 注意事项
1. 每一行都必须按照 `键:值` 的格式
2. 化学式要准确，系统会用它来匹配反应
3. 摩尔质量可以保留3位小数
4. 状态只能是：固体、液体、气体之一
5. 文件名可以任意命名，但扩展名必须是 `.txt`
6. 不要在模板文件中使用中文冒号（：），必须使用英文冒号（:）
