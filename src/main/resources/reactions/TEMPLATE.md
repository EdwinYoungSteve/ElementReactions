# 反应文件模板

## 文件格式说明
在 `src/main/resources/reactions/` 目录下创建 `.txt` 文件来定义化学反应。

## 文件格式
```
反应名称:反应的名称
反应物:化学式1,化学式2,...
反应物化学计量数:数字1,数字2,...
产物:化学式1,化学式2,...
产物化学计量数:数字1,数字2,...
反应条件:条件说明（如：点燃、加热、催化剂等）
反应类型:反应类型（如：化合反应、分解反应、置换反应等）
描述:反应的描述
```

## 示例1：氢氧燃烧反应
```
反应名称:氢氧燃烧反应
反应物:H2,O2
反应物化学计量数:2,1
产物:H2O
产物化学计量数:2
反应条件:点燃
反应类型:化合反应
描述:氢气在氧气中燃烧生成水
```

对应的化学方程式：2H2 + O2 ―[点燃]→ 2H2O

## 示例2：钠与氯气反应
```
反应名称:钠与氯气反应
反应物:Na,Cl2
反应物化学计量数:2,1
产物:NaCl
产物化学计量数:2
反应条件:点燃
反应类型:化合反应
描述:钠在氯气中燃烧生成氯化钠
```

对应的化学方程式：2Na + Cl2 ―[点燃]→ 2NaCl

## 注意事项
1. 每一行都必须按照 `键:值` 的格式
2. 多个化学式用逗号分隔
3. 化学计量数的顺序必须与化学式的顺序对应
4. 确保反应中用到的所有化学式都在 `compounds` 文件夹中有对应的化合物文件
5. 文件名可以任意命名，但扩展名必须是 `.txt`
