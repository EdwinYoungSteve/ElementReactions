# 快速开始指南

## 运行程序

编译后直接运行可执行文件：
```bash
./ElementReactions    # Linux/Mac
ElementReactions.exe  # Windows
```

## 基本操作

### 1. 查看可用试剂
程序启动后会自动显示试剂库中的所有化合物。

### 2. 选择试剂进行反应
输入试剂编号，用空格分隔。例如：
- 输入 `1 2` 选择第1和第2种试剂
- 输入 `4 5` 选择第4和第5种试剂

### 3. 退出程序
输入 `0` 退出程序。

## 自定义反应

### 步骤1：添加化合物
在 `src/main/resources/compounds/` 目录创建文件 `carbon.txt`：
```
名称:碳
化学式:C
摩尔质量:12.011
状态:固体
描述:黑色固体
```

### 步骤2：添加反应
在 `src/main/resources/reactions/` 目录创建文件 `carbon_oxygen.txt`：
```
反应名称:碳的燃烧
反应物:C,O2
反应物化学计量数:1,1
产物:CO2
产物化学计量数:1
反应条件:点燃
反应类型:化合反应
描述:碳在氧气中充分燃烧
```

### 步骤3：重启程序
重新运行程序，新的化合物和反应会自动加载。

## 常见问题

### Q: 为什么选择的试剂没有发生反应？
A: 可能的原因：
1. 反应库中没有定义这个反应
2. 化学式拼写错误
3. 化合物文件和反应文件中的化学式不匹配

### Q: 如何知道系统支持哪些反应？
A: 当输入的试剂无法匹配任何反应时，系统会列出所有可用的反应。

### Q: 可以同时添加多种试剂吗？
A: 可以，但目前系统只支持匹配反应库中定义的反应。如果需要三种或更多试剂的反应，需要在反应文件中明确定义。

## 提示

1. 所有文本文件使用 UTF-8 编码
2. 使用英文冒号（:）而不是中文冒号（：）
3. 化学式区分大小写（H2O 正确，h2o 错误）
4. 每次修改文件后需要重启程序才能生效
